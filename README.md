## Virtual Machine(VM) Translator
This Virtual Machine to HACK assembly translator was done as part of Project 6 of my [nand2tetris](https://github.com/paudsu01/nand2tetris) course. <br>
This VMTranslator translates VM Code to HACK assembly language creating a `.vm` file.
